# Content Model

## Core Records
- article
- article translation
- category
- category translation
- media file
- visitor comment
- user role
- audit entry

## Language Strategy
The content structure is prepared for:
- Turkish
- English
- Russian

## Publishing Flow
1. Draft is created
2. Translation records are added
3. Media is attached
4. Review is completed
5. Publish status is enabled

## Moderation Flow
1. Visitor comment is submitted
2. Review queue receives the entry
3. Comment is approved or removed
