# Project Scope

## Purpose
namikgedik.com will be a public-facing technology website for articles, news, and media content.

## Core Goals
- Publish technology articles, images, and videos
- Support Turkish, English, and Russian languages
- Allow visitors to comment with name and email
- Provide an admin panel for content management
- Keep the system suitable for IIS-based deployment

## Planned Modules
1. Public website
2. Admin panel
3. Backend API
4. Database layer
5. Media/content management
6. Multi-language structure
7. SEO and analytics basics

## MVP Priorities
- Homepage
- Article listing
- Article detail page
- Category structure
- Basic admin login
- Article create/edit/delete
- Comment moderation
- TR / EN / RU content structure

## Future Phase
- Membership system
- Saved articles / favorites
- Newsletter integration
- Rich search and filtering
- Role-based management
